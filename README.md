# Health Management System

## About Project

Health Management System is a web-based application developed to manage healthcare information.

The system helps manage doctors, patients, appointments and medical records.

## Technologies Used

- PHP
- MySQL
- HTML
- CSS
- JavaScript

## Features

- Patient management
- Doctor management
- Appointment booking
- Medical records
- Database management

## Project Structure
Health-Management-System
|
|-- admin
|-- doctor
|-- patient
|-- css
|-- js
|-- database


## Installation

1. Install XAMPP
2. Copy project into htdocs
3. Start Apache and MySQL
4. Import database
5. Open browser
